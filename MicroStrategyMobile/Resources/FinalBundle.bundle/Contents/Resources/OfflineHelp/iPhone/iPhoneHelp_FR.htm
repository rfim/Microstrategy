Les informations suivantes sont une présentation complète des fonctions de base de MicroStrategy Mobile pour iPhone.

Écran d'accueil

Lorsque vous ouvrez MicroStrategy Mobile sur votre iPhone, l'écran d'accueil s'affiche. L'écran d'accueil se compose de plusieurs boutons permettant d'accéder à différentes parties de l'application Mobile.

Les boutons affichés sur l'écran d'accueil peuvent varier en fonction de la manière dont votre administrateur a configuré MicroStrategy Mobile. Pour plus d'informations sur la personnalisation de l'écran d'accueil, reportez-vous à l'aide installée avec Mobile Server Administrator.

La première fois que vous installez l'application à partir de l'App Store, les boutons de l'écran d'accueil vous permettent d'accéder rapidement à des exemples de données. Une fois que vous avez configuré l'application afin d'accéder à votre système MicroStrategy, l'écran d'accueil par défaut comporte les options suivantes :

  • Rapports : Affiche tous les rapports et documents auxquels vous avez souscrit. Remuez le périphérique pour mettre la liste à jour. Appuyez brièvement sur le rapport ou le document pour l'exécuter. Pour plus d'informations sur la souscription à un rapport ou un document, reportez-vous à l'aide de MicroStrategy.
  • Bibliothèque partagée : Affiche tous les projets auxquels vous avez accès. Appuyez brièvement sur un projet pour afficher les dossiers qu'il contient. Appuyez brièvement sur un dossier pour afficher les rapports/documents qu'il contient. Appuyez brièvement sur un rapport/document pour l'exécuter.
  • Paramètres : Ouvre l'écran Paramètres. En fonction de la configuration de votre périphérique, cet écran vous permet de modifier les informations de connexion, les informations de serveur et les options générales d'administration et d'affichage de l'application.
  • Aide : Affiche ce fichier d'aide.

Exécution des rapports et des documents

Lorsque vous affichez la liste des rapports/documents, les rapports de grille, les rapports graphiques et les documents sont identifiés par des icônes.

Pour exécuter un rapport/document, recherchez-le sur l'écran d'accueil et appuyez brièvement dessus. Le rapport/document s'ouvre.

Réponse aux invites

Certains rapports/documents nécessitent des réponses à des questions préalables, ou à des invites, pour que les données puissent s'afficher. La procédure de réponse aux invites dépend des informations requises et du type d'invite.

Analyse des données

Lorsqu'un rapport/document est exécuté, il existe plusieurs façons d'interagir avec pour faciliter l'analyse.

Lorsqu'un rapport/document est exécuté, appuyez brièvement dessus pour passer du mode Plein écran au mode normal.

En mode normal, un en-tête affiche le bouton Précédent, le nom du rapport/document et le groupe de données actuellement affiché. Appuyez brièvement sur le bouton Précédent pour revenir à la liste des rapports/documents.

Appuyez brièvement sur les mesures affichées dans l'en-tête pour qu'elles s'affichent ou non dans le rapport/document.

Les données d'entreprise peuvent être groupées (dans un saut de page) en haut du rapport/document. Appuyez brièvement sur le groupe de données actuellement affiché et choisissez un autre groupe ou effectuez un balayage horizontal sur le bord de l'écran pour passer d'un groupe à un autre.

Pour agrandir ou réduire la visualisation d'un rapport ou d'un document, effectuez un pincement.

Certains rapports/documents peuvent s'afficher au format paysage ou portrait. Selon la manière dont le rapport/document a été conçu, vous pouvez modifier cette orientation en faisant pivoter votre périphérique.

Certains rapports/documents vous permettent de consulter des informations à plusieurs niveaux. Par exemple, par défaut un rapport peut afficher le temps au niveau de l'année, mais il peut aussi afficher les données groupées par trimestre ou par mois. Si tel est le cas, l'en-tête Année sera souligné. Pour modifier le niveau auquel les données sont affichées, appuyez brièvement sur un en-tête souligné.
