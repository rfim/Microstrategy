Die folgenden Informationen bieten einen umfassenden Überblick über die grundlegenden Funktionen von MicroStrategy Mobile für iPhone.

Der Startseite-Bildschirm

Wenn Sie MicroStrategy Mobile auf Ihrem iPhone öffnen, wird der Startseite-Bildschirm angezeigt. Der Startseite-Bildschirm besteht aus mehreren Schaltflächen, die verschiedene Bereiche der Mobile-Anwendung öffnen.

Die auf dem Startseite-Bildschirm angezeigten Schaltflächen können entsprechend der Konfiguration von MicroStrategy Mobile durch Ihren Administrator variieren. Weitere Informationen zur Anpassung des Startseite-Bildschirms finden Sie in der Hilfe, die zusammen mit Mobile Server Administrator installiert wurde.

Wird die Anwendung erstmalig vom App Store installiert, ermöglichen die Schaltflächen des Startseite-Bildschirms schnellen Zugriff auf Beispieldaten. Haben Sie die Anwendung zum Zugreifen auf Ihr MicroStrategy-System konfiguriert, setzt sich der standardmäßige Startseite-Bildschirm aus den folgenden Optionen zusammen:

  • Berichte : zeigt alle von Ihnen abonnierten Berichte und Dokumente an. Schütteln Sie das Gerät, wird die Liste aktualisiert. Tippen Sie auf einen Bericht oder ein Dokument, um ihn/es auszuführen. Weitere Informationen zum Abonnieren eines Berichts oder Dokuments finden Sie in der Hilfe von MicroStrategy.
  • Öffentliche Bibliothek : zeigt alle Projekte, auf die Sie Zugriff haben. Tippen Sie auf ein Projekt, werden die darin enthaltenen Ordner angezeigt. Tippen Sie auf einen Ordner, werden die darin enthaltenen Ordner und Berichte/Dokumente angezeigt. Tippen Sie auf einen Bericht/ein Dokument, so wird dieser/dieses ausgeführt.
  • Einstellungen : öffnet den Bildschirm „Einstellungen“. Abhängig von Ihrer Gerätekonfiguration können Sie hier Ihre Informationen zu Anmeldedaten, Server-Informationen und allgemeine administrative und Anzeigeoptionen für die Anwendung ändern.
  • Hilfe : zeigt die Hilfedatei an.

Ausführen von Berichten und Dokumenten

Wenn Sie die Liste der Berichte/Dokumente anzeigen, werden Tabellenberichte, Diagrammberichte und Dokumente anhand von Symbolen identifiziert.

Um einen Bericht/ein Dokument auszuführen, suchen Sie im Startseite-Bildschirm danach und tippen darauf. Der Bericht/das Dokument wird geöffnet.

Beantworten von Eingabeaufforderungen

Bei einigen Berichten/Dokumenten müssen zunächst Fragen - oder Eingaben - beantwortet werden, bevor die Daten angezeigt werden können. Die notwendigen Schritte zur Beantwortung von Eingaben hängen von den benötigten Informationen und dem Typ der Eingabe ab.

Analysieren von Daten

Wurde ein Bericht/Dokument ausgeführt, gibt es verschiedene Möglichkeiten, damit zu interagieren, um die Analyse zu vereinfachen.

Nachdem ein Bericht/ein Dokument ausgeführt wurde, tippen Sie darauf, um zwischen dem Vollbildmodus und dem Nicht-Vollbildmodus zu wechseln.

Im Nicht-Vollbildmodus wird in der Kopfzeile eine Zurück-Schaltfläche, der Berichts-/Dokumentname und die aktuell dargestellte Datengruppe angezeigt. Tippen Sie auf die Zurück-Schaltfläche, kehren Sie zur Liste der Berichte/Dokumente zurück.

Tippen Sie auf andere Metriken, die in der Kopfzeile angezeigt werden, wird zwischen deren Anzeige im Bericht/Dokument umgeschaltet.

Unternehmensdaten können u. U. über den oberen Bereich eines Berichts/Dokuments gruppiert (in einem Page-By) sein. Tippen Sie auf die aktuell angezeigte Gruppierung an Daten und wählen Sie eine andere Gruppierung oder ziehen Sie Ihren Finger horizontal über den Bildschirmrand, um zwischen Gruppierungen umzuschalten.

Um einen Bericht oder ein Dokument zu vergrößern/verkleinern, ziehen Sie es gleichzeitig mit Daumen und Zeigefinger („Pinch“).

Einige Berichte/Dokumente können entweder in Hoch- oder Querformat eingesehen werden. Abhängig davon, wie der Bericht/das Dokument gestaltet wurde, können Sie diese Ausrichtung ändern, indem Sie Ihr Gerät drehen.

Bei einigen Berichten/Dokumenten können Sie Informationen auf mehreren Ebenen einsehen. Beispielsweise zeigt ein Bericht standardmäßig die Zeit auf der Jahresebene an; dieser Bericht könnte die Daten jedoch auch nach Quartal oder Monat gruppiert anzeigen. In diesem Fall wird die Überschrift „Jahr“ unterstrichen angezeigt. Um die Ebene, auf der Sie Daten einsehen, zu ändern, tippen Sie auf eine unterstrichene Überschrift.
