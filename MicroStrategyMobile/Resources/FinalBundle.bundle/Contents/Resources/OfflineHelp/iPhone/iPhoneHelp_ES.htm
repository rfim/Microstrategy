La siguiente información es un vistazo a grandes rasgos de MicroStrategy Mobile sobre las funciones básicas para iPhone.

Pantalla de inicio

Cuando abre MicroStrategy Mobile en su iPhone, aparece la pantalla de inicio. La pantalla de inicio consiste de varios botones que abren diferentes partes de la aplicación Mobile.

Los botones que se muestran en la pantalla inicial pueden variar según cómo su administrador haya configurado MicroStrategy Mobile. Si desea obtener información adicional acerca de la pantalla de inicio, consulte la ayuda instalada con Mobile Server Administrator.

Cuando se instala la aplicación por primera vez desde App Store, los botones de la pantalla de inicio permiten obtener acceso rápido a los datos de ejemplo. Una vez que haya configurado la aplicación para que tenga acceso al sistema MicroStrategy, la pantalla de inicio por defecto constará de las siguientes opciones:

  • Informes : muestra todos los informes y documentos a los que está suscrito. Al mover el dispositivo, se actualiza la lista. Toque un informe o documento para ejecutarlo. Para obtener más información sobre la suscripción a un informe o documento, consulte la MicroStrategy Help.
  • Biblioteca compartida (Shared Library) : muestra todos los proyectos a los que tiene acceso. Al tocar un proyecto, se muestran las carpetas que contiene. Al tocar una carpeta se muestran las carpetas y los informes o documentos que contiene. Al tocar un informe o documento, se ejecuta.
  • Configuración (Settings) : abre la pantalla Configuración. Según la configuración del dispositivo, este le permitirá cambiar la información de inicio de sesión, la información del servidor y las opciones generales administrativas y de pantalla relativas a la aplicación.
  • Ayuda : muestra este archivo Ayuda.

Ejecución de informes y documentos

Cuando se visualiza la lista de informes o documentos, los informes de cuadrícula y los documentos se identifican con iconos.

Para ejecutar un informe o documento, navegue hasta su ubicación, desde la pantalla de inicio, y tóquelo. El documento o informe se abrirá.

Respuesta a selecciones dinámicas

Algunos informes o documentos requieren que el usuario responda a preguntas o selecciones dinámicas preliminares antes de que se puedan mostrar los datos. Los pasos requeridos para responder a las selecciones dinámicas dependen de la información solicitada y del tipo de selección dinámica.

Análisis de datos

Una vez que se ejecutó el informe o documento, existen varias maneras de interactuar con el informe o documento para facilitar el análisis.

Después de que haya ejecutado el informe o documento, tóquelo para alternar entre el modo de pantalla completa y parcial.

En modo de pantalla parcial, un encabezado muestra el botón Atrás, el nombre del informe o documento, y el grupo de datos mostrado actualmente. Si toca el botón Atrás, regresa a la lista de informes o documentos.

Si toca los diferentes indicadores mostrados en el encabezado, alternará entre uno y otro ya sea que aparezcan o no en el informe o documento.

Los datos comerciales pueden agruparse (en una página) a lo largo de la parte superior de un informe o documento. Toque el grupo de datos mostrado actualmente y elija otro grupo o realice un voleo horizontal en el borde de la pantalla para moverse entre los grupos.

Para aumentar o reducir el tamaño de un informe o documento, realice el gesto de un pellizco.

Algunos informes o documentos se pueden visualizar con orientación vertical u horizontal. Según cómo haya sido diseñado el documento o informe, puede cambiar esta orientación haciendo girar el dispositivo.

Algunos informes o documentos le permiten ver información en varios niveles. Por ejemplo, un informe puede mostrar la hora a nivel del año por defecto, pero puede mostrar datos agrupados por trimestre o mes. Si este es el caso, el encabezado Año aparecerá subrayado. Para cambiar el nivel en el que visualiza los datos, toque el encabezado subrayado.
