Le informazioni riportate di seguito offrono una panoramica generale delle funzioni di base di MicroStrategy Mobile per iPhone.

Schermata Home

All'apertura di MicroStrategy Mobile nell'iPhone viene visualizzata la schermata Home. Tale schermata contiene vari pulsanti che consentono di aprire i diversi componenti dell'applicazione Mobile.

I pulsanti visualizzati nella schermata Home cambiano in base al tipo di configurazione di MicroStrategy Mobile scelta dall'amministratore. Per ulteriori informazioni sulla personalizzazione della schermata Home, consultare la Guida installata con Mobile Server Administrator.

Quando si installa l'applicazione per la prima volta dall'App Store, i pulsanti della schermata Home consentono di accedere rapidamente ai dati di esempio. Dopo aver configurato l'applicazione per accedere al sistema MicroStrategy, la schermata Home predefinita è costituita dalle seguenti opzioni:

  • Report : visualizza tutti i report e i documenti per i quali si dispone della sottoscrizione. Scuotere il dispositivo per aggiornare l'elenco. Toccare un report o documento per eseguirlo. Per ulteriori informazioni sulla sottoscrizione ad un report o documento, consultare la Guida di MicroStrategy.
  • Libreria condivisa : visualizza tutti i progetti ai quali è possibile accedere. Toccare un progetto per visualizzare le cartelle in esso contenute. Toccare una cartella per visualizzare le cartelle e i report/documenti in essa contenuti. Toccare un report/documento per eseguirlo.
  • Impostazioni: apre la schermata Impostazioni. A seconda della configurazione del dispositivo è possibile modificare le informazioni di login, quelle relative al server e le opzioni generali di amministrazione e visualizzazione per l'applicazione.
  • Guida : visualizza questo file della Guida.

Esecuzione di report e documenti

Nell'elenco dei report o documenti, i report tabella, i report grafici e i documenti sono identificati da icone.

Per eseguire un report o documento, nella schermata Home toccare l'icona corrispondente. Il report o il documento si apre.

Risposta ai prompt

Per alcuni report o documenti è necessario rispondere a domande, o prompt, preliminari prima di poter visualizzare i dati. Il numero di passaggi varia a seconda delle informazioni richieste e del tipo di prompt.

Analisi dei dati

Una volta eseguito un report o documento, esistono diversi metodi per semplificare l'analisi.

Dopo l'esecuzione di un report o documento, toccarlo per alternare tra la modalità a schermo intero e quella non a schermo intero.

Nella modalità non a schermo intero un'intestazione mostra il pulsante Indietro, il nome del report o documento e il gruppo di dati attualmente visualizzato. Toccare il pulsante Indietro per tornare all'elenco di report o documenti.

Toccare metriche diverse mostrate nell'intestazione per visualizzarle sul report o documento o per nasconderle.

I dati aziendali possono essere raggruppati (in pagine) lungo la parte superiore del report o documento. Toccare il gruppo di dati visualizzato e selezionare un altro gruppo o sfiorare orizzontalmente il bordo della schermata per spostarsi da un gruppo all'altro.

Pizzicare per eseguire uno zoom in avanti o indietro su un report o documento.

Alcuni report o documenti possono essere visualizzati con un orientamento orizzontale o verticale. A seconda del modo in cui è stato realizzato il report o documento, è possibile cambiare questo orientamento ruotando il dispositivo.

Alcuni report o documenti consentono di visualizzare le informazioni a più livelli. Ad esempio, in un report i dati che per impostazione predefinita sono visualizzati per anno possono essere raggruppati per trimestre o per mese. In tal caso, l'intestazione Anno appare sottolineata. Per cambiare il livello di visualizzazione dei dati, toccare un'intestazione sottolineata.
