以下信息是通过 MicroStrategy Mobile 了解 iPhone 基本功能的高级查看方式。

主屏幕

当您在 iPhone 上打开 MicroStrategy Mobile 时，将显示主屏幕。主屏幕由几个可打开 Mobile 应用程序的不同组件的按钮组成。

根据您的管理员对 MicroStrategy Mobile 进行的配置，主屏幕上所显示的按钮会有所不同。如需了解更多关于自定义主屏幕的信息，请参阅与 Mobile Server Administrator 一起安装的“帮助”。

应用程序首次通过 App Store 安装时,主屏幕按钮允许快速访问示例数据。对应用程序进行配置使其可访问 MicroStrategy 系统后，默认的主屏幕将由以下选项组成：

  • 报表 ：显示您已订阅的所有报表和文档。摇晃设备即可更新列表。轻按报表或文档以执行。如需更多关于订阅报表或文档的信息，请参阅 MicroStrategy Help。
  • 共享库 ：显示所有您可访问的项目。轻按项目即可显示项目内的文件夹。轻按文件夹可显示该文件夹内的子文件夹以及报表/文档。轻按报表/文档来执行它。
  • 设置 ：打开“设置”屏幕。根据您设备配置的不同，可允许您更改您的登录信息、服务器信息、以及应用程序的常规管理和显示选项。
  • 帮助 ：显示此“帮助”文件。

运行报表和文档

当查看报表/文档列表时，表格式报表、图表式报表以及文档将以图标加以识别。

要运行一个报表/文档，从主屏幕浏览查找并轻按它。这时报表/文档将被打开。

回答提示

部分报表/文档在显示数据之前要求用户必须回答初始问题或提示。回答提示所必须的步骤取决于所请求的信息和提示类型。

分析数据

报表/文档运行后，您可以通过几种方式与其进行交互操作从而让分析变得更简单。

报表/文档运行后，通过轻按操作可以在全屏模式和非全屏模式之间进行切换。

在非全屏模式下，标题中显示后退按钮、报表/文档名称以及当前所显示的数据组。轻按后退按钮可以回到报表/文档列表。

轻按标题中所显示的不同度量可以切换是否在报表/文档中显示这些度量。

业务数据可在报表/文档的顶部进行分组（位于一个分页）。轻按当前显示的数据组并选择其他分组，或在屏幕边缘进行水平滑动操作以便在分组间移动。

要缩放报表或文档，可进行划屏操作。

部分报表/文档既可以横向查看也纵向查看。根据报表/文档的设计，您可以通过旋转设备来更改显示方向。

部分报表/文档允许您在多个级别查看信息。例如，报表可以默认在年级别来显示时间，但也可以显示按季度和月份分组的数据。在这种情况下，标题 Year（年份）将带下划线显示。要更改您查看数据所在的级别，轻按带下划线的标题。
