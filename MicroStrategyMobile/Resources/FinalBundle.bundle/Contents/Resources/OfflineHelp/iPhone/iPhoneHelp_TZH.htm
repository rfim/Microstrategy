以下資訊是透過 MicroStrategy Mobile 瞭解 iPhone 基本功能的進階檢視方式。

主畫面

當您在 iPhone 上打開 MicroStrategy Mobile 時，將顯示主畫面。主畫面由幾個可打開 Mobile 應用程式的不同元件的按鈕組成。

根據您的管理員對 MicroStrategy Mobile 進行的設定，主畫面上所顯示的按鈕會有所不同。如需瞭解更多關於自訂主畫面的資訊，請參閱與 Mobile Server Administrator 一起安裝的“說明”。

應用程式首次透過 App Store 安裝時,主畫面按鈕允許快速訪問範例資料。對應用程式進行設定使其可訪問 MicroStrategy 系統後，預設的主畫面將由以下選項組成：

  • 報表 ：顯示您已訂閱的所有報表和檔案。搖晃裝置即可更新清單。觸碰報表或檔案以執行。如需更多關於訂閱報表或檔案的資訊，請參閱 MicroStrategy Help。
  • 共用庫 ：顯示所有您可訪問的專案。觸碰專案即可顯示專案內的資料夾。觸碰資料夾可顯示該資料夾內的子資料夾以及報表/檔案。觸碰報表/檔案來執行它。
  • 設定 ：打開“設定”螢幕。根據您裝置設定的不同，可允許您變更您的登入資訊、伺服器資訊、以及應用程式的一般管理和顯示選項。
  • 説明 ：顯示此“説明”檔案。

執行報表和檔案

當檢視報表/檔案清單時，表格式報表、圖表式報表以及檔案將以圖示加以識別。

要執行一個報表/檔案，從主畫面瀏覽尋找並觸碰它。這時報表/檔案將被打開。

回答提示

部分報表/檔案在顯示資料之前要求使用者必須回答初始問題或提示。回答提示所必須的步驟取決於所請求的資訊和提示類型。

分析資料

報表/檔案執行後，您可以透過幾種方式與其進行互動操作從而讓分析變得更簡單。

報表/檔案執行後，透過觸碰操作可以在全螢幕模式和非全螢幕模式之間進行切換。

在非全螢幕模式下，標題中顯示後退按鈕、報表/檔案名稱以及目前所顯示的資料組。觸碰後退按鈕可以回到報表/檔案清單。

觸碰標題中所顯示的不同度量可以切換是否在報表/檔案中顯示這些度量。

業務資料可在報表/檔案的頂部進行群組（位於一個分頁）。觸碰目前顯示的資料組並選取其他群組，或在螢幕邊緣進行水平滑動操作以便在群組間移動。

要縮放報表或檔案，可進行劃屏操作。

部分報表/檔案既可以橫向檢視也縱向檢視。根據報表/檔案的設計，您可以透過旋轉裝置來變更顯示方向。

部分報表/檔案允許您在多個級別檢視資訊。例如，報表可以默認在年級別來顯示時間，但也可以顯示按季度和月份群組的資料。在這種情況下，標題 Year（年份）將帶底線顯示。要變更您檢視資料所在的級別，觸碰帶底線的標題。
