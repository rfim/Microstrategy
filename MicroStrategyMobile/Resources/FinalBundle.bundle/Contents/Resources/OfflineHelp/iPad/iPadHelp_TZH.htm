以下資訊是透過 MicroStrategy Mobile 瞭解 iPad 基本功能的進階檢視方式。

主畫面

當您在 iPad 上打開 MicroStrategy Mobile 時，將顯示主畫面。主畫面由幾個可打開 Mobile 應用程式的不同元件的圖示組成。

根據您的管理員對 MicroStrategy Mobile 進行的設定，主畫面上所顯示的按鈕會有所不同。

預設的主畫面由以下幾個索引標籤和按鈕組成：

  • 首頁：顯示您可訪問的所有專案。
  • 報表 ：顯示您已訂閱的所有報表和文件。觸碰報表或檔案以執行。
  • “ i ”按鈕 ：允許您訪問“設定”螢幕，以及此“說明”檔案。
  • 重新整理按鈕 ：重新整理“主頁”索引標籤中的專案清單，以及重新整理“報表”索引標籤中的訂閱。

執行報表和檔案

當檢視報表/文件清單時，表格式報表、圖表式報表以及文件將以縮圖預覽加以識別。如果預覽圖丟失，將顯示表格、圖表或文件的普通圖示。

要執行一個報表/文件，從主畫面瀏覽尋找並觸碰它的圖示。這時將打開報表或文件。

回答提示

部分報表/檔案在顯示資料之前要求使用者必須回答初始問題或提示。回答提示所必須的步驟取決於所請求的資訊和提示類型。

列印報表和文件

如果您的 iPad 執行的系統是 iOS 4.2 以上的版本，您可以透過 MicroStrategy Mobile 列印報表和文件。

  1. 檢視報表或文件的同時，輕按 操作 按鈕： 。
  2. 觸碰 列印 。這時將打開 印表機選項 對話方塊。
  3. 要從您的網路中選取印表機，觸碰 印表機 。
  4. 要選取列印頁面，觸碰 頁面範圍 。
  5. 要變更副本的數量，觸碰加號或減號按鈕。
  6. 當列印就緒時，觸碰列印。

檢視並分析資料

報表/檔案執行後，您可以透過幾種方式與其進行互動操作從而讓分析變得更簡單。

業務資料可在報表/檔案的頂部進行群組（位於一個分頁）。觸碰目前顯示的資料組並選取其他群組，或執行以下筆勢以便在頁面之間移動：

  • 左邊開始第一個群組：向右或向左滑動您的指尖分別切換至下一頁或上一頁。
  • 左邊開始第二個群組：向上或向下滑動您的指尖分別切換至下一頁或上一頁。

要檢視儲存格中資料的放大視圖，觸碰並將您的指尖停留在儲存格上。朝其他儲存格方向拖動指尖來放大它們。

部分報表/檔案允許您在多個級別檢視資訊。例如，報表可以默認在年級別來顯示時間，但也可以按季度或月份，或者按“城市”等其他維度顯示群組資料。如果報表支援此功能，觸碰並將您的指尖停留在維度標題或元素上以打開上下文功能表。觸碰 任意位置鑽取 。這時將出現一個圖形鑽取路徑選取器。要選取被鑽取的維度，捲動清單直至您要鑽取的維度出現在螢幕中央位置。觸碰該維度以檢視其中的維度清單。選取一個風格類似的維度。完成後，觸碰 更新 。
