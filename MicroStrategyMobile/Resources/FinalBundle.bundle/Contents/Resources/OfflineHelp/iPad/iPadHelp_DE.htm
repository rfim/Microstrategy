Die folgenden Informationen bieten einen umfassenden Überblick über die grundlegenden Funktionen von MicroStrategy Mobile für iPad.

Der Startseite-Bildschirm

Wenn Sie MicroStrategy Mobile auf Ihrem iPad öffnen, wird der Startseite-Bildschirm angezeigt. Der Startseite-Bildschirm besteht aus mehreren Symbolen, die verschiedene Bereiche der Mobile-Anwendung öffnen.

Die auf dem Startseite-Bildschirm angezeigten Schaltflächen können entsprechend der Konfiguration von MicroStrategy Mobile durch Ihren Administrator variieren.

Der standardmäßige Startseite-Bildschirm besteht aus den folgenden Registerkarten und Schaltflächen:

  • Startseite: zeigt alle Projekte, auf die Sie Zugriff haben.
  • Berichte : zeigt alle von Ihnen abonnierten Berichte und Dokumente an. Tippen Sie auf einen Bericht oder ein Dokument, um ihn/es auszuführen.
  • „ i “-Schaltfläche : ermöglicht Ihnen den Zugriff auf den Bildschirm „Einstellungen“ und diese Hilfe-Datei.
  • Schaltfläche „Aktualisieren“ : aktualisiert die Liste an Projekten auf der Registerkarte „Startseite“ und aktualisiert Ihre Abonnements auf der Registerkarte „Berichte“.

Ausführen von Berichten und Dokumenten

Wenn Sie die Liste der Berichte/Dokumente anzeigen, werden Tabellenberichte, Diagrammberichte und Dokumente anhand einer Miniaturvorschau identifiziert. Fehlt die Vorschau, wird ein Symbol für eine Tabelle, ein Diagramm oder Dokument angezeigt.

Um einen Bericht/ein Dokument auszuführen, suchen Sie im Startseite-Bildschirm danach und tippen auf dessen Symbol. Der Bericht oder das Dokument wird geöffnet.

Beantworten von Eingabeaufforderungen

Bei einigen Berichten/Dokumenten müssen zunächst Fragen - oder Eingaben - beantwortet werden, bevor die Daten angezeigt werden können. Die notwendigen Schritte zur Beantwortung von Eingaben hängen von den benötigten Informationen und dem Typ der Eingabe ab.

Drucken von Berichten und Dokumenten

Verfügt Ihr iPad über iOS 4.2 oder höher, können Sie Berichte und Dokumente aus MicroStrategy Mobile heraus drucken.

  1. Während Sie einen Bericht oder ein Dokument einsehen, tippen Sie die Schaltfläche Aktion an: .
  2. Tippen Sie Drucken an. Das Dialogfeld Druckeroptionen wird geöffnet.
  3. Um einen Drucker aus Ihrem Netzwerk auszuwählen, tippen Sie Drucker an.
  4. Um die zu druckenden Seiten auszuwählen, tippen Sie Bereich an.
  5. Um die Anzahl an Kopien zu ändern, tippen Sie die Plus- oder Minus-Schaltflächen an.
  6. Sind Sie zum Drucken bereit, tippen Sie Drucken an.

Anzeigen und Analysieren von Daten

Wurde ein Bericht/Dokument ausgeführt, gibt es verschiedene Möglichkeiten, damit zu interagieren, um die Analyse zu vereinfachen.

Unternehmensdaten können u. U. über den oberen Bereich eines Berichts/Dokuments gruppiert (in einem Page-By) sein. Tippen Sie auf die aktuell angezeigte Gruppierung an Daten und wählen Sie eine andere Gruppierung, um zwischen Seiten zu wechseln.

Um eine vergrößerte Ansicht von Daten in einer Zelle einzusehen, tippen Sie eine Zelle an und halten Ihren Finger über der Zelle. Ziehen Sie Ihren Finger über andere Zellen, um sie zu vergrößern.

Bei einigen Berichten/Dokumenten können Sie Informationen auf mehreren Ebenen einsehen. Beispielsweise zeigt ein Bericht standardmäßig die Zeit auf der Jahresebene an; dieser Bericht könnte die Daten jedoch auch nach Quartal oder Monat oder anderen Dimensionen, wie z. B. Stadt, gruppiert anzeigen. Unterstützt der Bericht dies, tippen Sie eine Attributkopfzeile oder ein Element an und halten den Finger darüber, um das Kontextmenü zu öffnen. Tippen Sie Drill anywhere an. Ein graphischer Drill-Pfad-Selektor wird eingeblendet. Um eine Dimension, zu der die Drill-Funktion ausgeführt werden soll, auszuwählen, blättern Sie durch die Liste, bis sich die von Ihnen gewünschte Liste in der Mitte befindet. Tippen Sie die Dimension an, um eine Liste an Attributen darin einzusehen. Wählen Sie ein Attribut auf die gleiche Art aus. Sind Sie fertig, tippen Sie Aktualisieren an.
